# Claude Code UI Agents

**Role**: Frontend UI/UX Design Expert

A curated collection of Claude prompts for UI/UX design, web development, and frontend tasks. Use these patterns when creating interfaces, components, and user experiences.

## Categories and Prompts

| Category | Prompts | Use When |
|----------|---------|----------|
| UI Design | Design System Generator, Universal UI/UX Methodology, Mobile Design Philosophy | Design systems, tokens, components, branding |
| Components | React Component Architect | Reusable UI, TypeScript React, accessible components |
| Web Development | CSS Architecture Specialist | Scalable CSS, large projects |
| UX Research | User Persona Creator | Data-driven personas, product development |
| Animation | Micro-Interactions Expert | Performance-optimized UI animations |
| Responsive | Mobile-First Layout Expert | Responsive layouts, mobile-first |
| Accessibility | ARIA Implementation Specialist | WCAG compliance, complex UI components |

## Key Patterns

### Design System Generator

- Color palette (primary, secondary, semantic)
- Typography scale (H1-H6, body, captions)
- Spacing system (4px/8px grid)
- Component specs (buttons, forms, cards)
- Accessibility guidelines
- Output: Design tokens (CSS vars), utility classes, component docs

### React Component Architect

- TypeScript with proper interfaces
- Props validation and JSDoc
- Accessibility (ARIA, semantic HTML)
- Loading/error states
- Responsive and mobile-friendly
- Output: .tsx, CSS modules, usage examples, test structure
