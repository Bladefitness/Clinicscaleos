# Design System Generator

**Category:** ui-design
**Tags:** #design-system #tokens #components #branding

## Prompt

```
I need you to create a comprehensive design system for my project with the following specifications:

BRAND CONTEXT:
- Brand name: [BRAND_NAME]
- Industry: [INDUSTRY_TYPE]
- Brand personality: [DESCRIBE_BRAND_TRAITS] (e.g., professional, playful, modern, trustworthy)
- Target audience: [DESCRIBE_TARGET_USERS]

DESIGN SYSTEM COMPONENTS NEEDED:

1. COLOR SYSTEM: Primary, secondary, semantic colors, light/dark mode
2. TYPOGRAPHY: Font recommendations, type scale, line heights
3. SPACING: Base unit, margin/padding scale, layout guidelines
4. COMPONENTS: Buttons, forms, cards, navigation with variations
5. ACCESSIBILITY: Contrast, focus indicators, text minimums

OUTPUT: Design tokens (JSON/CSS vars), utility classes, component docs, usage guidelines.
```
